接口文档

接口地址 https://api.ahfi.cn/api/randomText
返回格式 JSON/TEXT
请求方式 GET/POST
请求示例
https://api.ahfi.cn/api/randomText?type=MeiJuZhaiChao&format=json

请求参数说明：

名称 必填 类型 说明
type 是 string 指定文案，如：TongJinCheng、TianGou、Sentences、Morning、MeiJuZhaiChao、LoveWords、KFCV50、GoodNight
format 否 string 返回数据的类型，可选值为 json 或 text。默认值为 json。

返回参数说明：

名称 类型 说明
data.type string 返回的指定的随机文案类型
data.content string 返回的文案内容
